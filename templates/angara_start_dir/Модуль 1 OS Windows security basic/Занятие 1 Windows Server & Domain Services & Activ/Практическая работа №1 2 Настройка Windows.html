Часть 0. Основная информация

При работе с системой виртуализации мы будем сталкиваться с различными нюансами самих платформ виртуализации.
Например, для того, чтобы передать ctrl+alt+del в виртуальную систему необходимо использовать хоткеи, определенные по умолчанию. Стандартный способ вызовет окно ctrl+alt+del в вашей хостовой системе, а не в виртуальной. В Virtual Box по умолчанию это сочетание вызывается клавишами
right ctrl+delete. Сочетания клавиш можно изменить в парамертах программы виртуализации.
