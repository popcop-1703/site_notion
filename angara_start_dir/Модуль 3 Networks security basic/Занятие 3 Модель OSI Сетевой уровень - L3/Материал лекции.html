Протоколы L3

  • IP - протокол, позволяющий передавать весь трафик
  • ICMP - протокол, который используется для проверки доступности того или иного хоста (ping, trace)
  • IGMP - протокол для проверки маршрутов
  • DVMRP - протокол для проверки маршрутов
  • IPsec - протокол для VPN
  • RIP - протокол динамической маршрутизации
  • OSPF - протокол динамической маршрутизации
  • BGP - протокол динамической маршрутизации

Inter VLAN routing

  1. Traditional Inter-VLAN Routing
    (способ передачи трафика в локальной сети между различными VLAN с помощью одного маршрутизатора или Switch L3)
  1. Router-on-a-Stick Inter-VLAN Routing
    (когда маршрутизация трафика происходит с помощью роутера, подключенного к коммутатору с помощью канала, настроенного как 
    802.1Q транк)
  1. Multilayer Switch Inter-VLAN Routing
    (когда маршрутизация трафика происходит с помощью устройства уровня
    L3, к которому подключеный несколько коммутаторов L2)

Динамическая маршрутизация

RIP

Прокол маршрутизации на основе вектора расстояния. Строит маршруты на основе кратчайшего пути.
Сейчас используется крайне редко, только в маленьких организация,так как есть ограничения в 15 хопов между роутерами и нет отслеживания состояния канала.

Важные таймеры протокола:

Update Timer - частота отправки обдновлений протокола (30 сек)

Invalid Timer - отметка о недостежимости маршрута (180 сек)

Flush Timer - удаление маршрута, если долго нет ответа после 180сек (240 сек)

Holddown timer - для предотвращения петель (180 сек)

OSPF

Open Shortest Path Finder - протокол динамической маршрутизации, созданный в 1988 году, основанный на технологии link-state, использует алгоритм Дейкстры. Очень часто используется в современных организациях.
Ключевое отличие от RIP, в том, что каждый из маршрутизаторов смотрит не только на то сколько хопов нужно пройти, а еще и на
состояние самого канала передачи.

Так же есть разделение на зоны(AS), для сокращения нагрузки на CPU маршрутизаторов и сокращения размера LSDB(базы данных маршрутизации)

BGP

BGP (Border Gateway Protocol, протокол гранричного шлюза) - протокол динамической маршрутизации

Основные термины:

  • IGP - внутренний протокол маршрутизации
  • EGP - внешний протокол маршрутизации
  • AS - автономная система
  • transit AS - транзитная автономная система
  • Path - путь
  • PA - атрибуты пути
  • Neighbor, peer - соседи
