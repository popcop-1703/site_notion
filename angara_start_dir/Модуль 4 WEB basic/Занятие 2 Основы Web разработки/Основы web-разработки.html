Настройте LAMP стек.

В качестве сервера используйте виртуальную машину на базе UNIX.

Установите Apache и php:

Запустите сервер и проверьте в браузере его работоспособность.

По умолчанию файлы apache, которые предназначены для клиентов web-сервера лежат в /var/www/html

Но этот путь при необходимости можно изменить, настроив соответствующий конфигурационный файл:

nano /etc/apache2/sites-enabled/000-default.conf

Для обучения пока ничего менять не будем.

Так как LAMP мы настраиваем для наших тестов, то ограничьте доступ к нему только по localhost

Для этого необходимо изменить конфиг apache2
