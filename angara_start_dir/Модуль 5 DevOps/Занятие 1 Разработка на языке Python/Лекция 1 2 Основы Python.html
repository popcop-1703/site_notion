1. Синтаксис

1.1 Строки кода

Строки кода - это основные элементы программы Python. Они состоят из ключевых слов, операторов, значений и других элементов языка.

print("Привет, мир!")

1.2 Отступы

В Python отступы используются для определения блоков кода. Это важно для читаемости и корректной работы программы.

if x > 0:
    print("x больше нуля")
else:
    print("x меньше или равно нулю")

1.3 Комментарии

Комментарии - это строки кода, которые не выполняются интерпретатором Python. Они используются для пояснения кода и повышения его читаемости.

# Это комментарий

print(10)

1.4 Операторы

Операторы используются для выполнения различных операций над значениями.

Примеры:

  • Арифметические операторы: +, -, *, /, //, %
  • Логические операторы: and, or, not
  • Операторы сравнения: ==, !=, >, <, >=, <=
print(2+2)

1.5 Ключевые слова

Ключевые слова - это зарезервированные слова, которые имеют специальное значение в Python.

Примеры:

  • if - используется для определения условного оператора
  • for - используется для определения цикла for
  • def - используется для определения функции

2. Переменные

2.1 Определение переменных

Переменные - это именованные области памяти, которые используются для хранения данных.

x = 10
y = "Привет, мир!"
print(x)
print(y)

2.2 Типы данных

В Python существует несколько типов данных:

  • Целые числа: int
  • Вещественные числа: float
  • Строки: str
  • Списки: list
  • Кортежи: tuple
  • Словари: dict
  • Множества: set
x = 10 # int
y = 3.14 # float
z = "Привет, мир!" # str

2.3 Присвоение значений

Для присвоения значения переменной используется оператор =.

x = 10
y = "Привет, мир!"

2.4 Операторы сравнения

Операторы сравнения используются для сравнения двух значений.

Примеры:

  • ==: равно
  • !=: не равно
  • >: больше
  • <: меньше
  • >=: больше или равно
  • <=: меньше или равно
x = 10
y = 20

print(x == y) # False
print(x != y) # True
print(x > y) # False
print(x < y) # True
print(x >= y) # False
print(x <= y) # True

3. Операторы

3.1 Арифметические операторы

Арифметические операторы используются для выполнения математических операций над значениями.

Примеры:

  • +: сложение
  • -: вычитание
  • *: умножение
  • /: деление
  • //: целочисленное деление
  • %: остаток от деления
x = 10
y = 20

print(x + y) # 30
print(x - y) # -10
print(x * y) # 200
print(x / y) # 0.5
print(x // y) # 0
print(x % y) # 10

3.2 Логические операторы

Логические операторы используются для выполнения логических операций над значениями.

Примеры:

  • and: и
  • or: или
  • not: не
x = 10
y = 20

print(x > 0 and y > 0) # True
print(x > 0 or y > 0) # True
print(not x > 0) # False

3.3 Операторы присваивания

Операторы присваивания используются для присвоения значения переменной.

Примеры:

  • =: простое присваивание
  • +=: прибавление значения
  • -=: вычитание значения
  • *=: умножение значения
  • /=: деление значения
  • //=: целочисленное деление значения
  • %=: остаток от деления
x = 10

x += 5 # x = 15
x -= 3 # x = 12
x *= 2 # x = 24
x /= 4 # x = 6
x //= 2 # x = 3
x %= 3 # x = 0

4. Условные выражения

Условные выражения используются для выполнения различных действий в зависимости от условия.

4.1 Оператор if

Оператор if используется для проверки условия и выполнения кода, если условие истинно.

x = 10

if x > 0:
    print("x больше нуля")

4.2 Оператор else

Оператор else используется для выполнения кода, если ни одно из предыдущих условий не было истинно.

x = 0

if x > 0:
    print("x больше нуля")
else:
    print("x меньше нуля")

4.3 Оператор elif

Оператор elif используется для проверки следующего условия, если предыдущее условие оказалось ложным.

x = 0

if x > 0:
    print("x больше нуля")
elif x < 0:
    print("x меньше нуля")
else:
    print("x равно нулю")

4.4 Вложенные условные выражения

Условные выражения могут быть вложены друг в друга.

x = 10
y = 20

if x > 0:
    if y > 0:
        print("x и y больше нуля")
    else:
        print("x больше нуля, но y меньше или равно нулю")
else:
    print("x меньше или равно нулю")

5. Циклы

5.1 Цикл for

Цикл for используется для выполнения кода над последовательностью значений.

for i in range(10):
    print(i)

5.2 Цикл while

Цикл while используется для выполнения кода, пока условие истинно.

i = 0
while i < 10:
    print(i)
    i += 1

5.3 Вложенные циклы

Вложенные циклы - это циклы, которые находятся внутри других циклов.

for i in range(10):
    for j in range(5):
        print(i, j)

5.4 Оператор break

Оператор break используется для досрочного выхода из цикла.

for i in range(10):
    if i == 5:
        break
    print(i)

5.5 Оператор continue

Оператор continue используется для пропуска текущей итерации цикла.

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

6. Работа с данными

6.1 Списки

Списки - это упорядоченные коллекции данных, которые могут содержать элементы разных типов.

my_list = [1, 2, 3, "Hello", True]

6.1.1 Операции со списками

  • Добавление элементов:
    • list.append(element): добавляет элемент в конец списка.
    • list.extend(iterable): добавляет элементы из iterable в конец списка.
  • Удаление элементов:
    • list.remove(element): удаляет первое вхождение элемента из списка.
    • list.pop(index): удаляет элемент по индексу.
  • Сортировка:
    • list.sort(): сортирует список по возрастанию.
    • list.sort(reverse=True): сортирует список по убыванию.
  • Поиск элементов:
    • list.index(element): возвращает индекс первого вхождения элемента в списке.
    • list.count(element): возвращает количество вхождений элемента в списке.
my_list = [1, 2, 3]

my_list.append(4)  # [1, 2, 3, 4]
my_list.extend([5, 6])  # [1, 2, 3, 4, 5, 6]

my_list.remove(3)  # [1, 2, 4, 5, 6]
my_list.pop(1)  # [1, 4, 5, 6]

my_list.sort()  # [1, 4, 5, 6]
my_list.sort(reverse=True)  # [6, 5, 4, 1]

print(my_list.index(4))  # 2
print(my_list.count(5))  # 1

6.2 Кортежи

Кортежи - это неизменяемые упорядоченные коллекции данных, которые могут содержать элементы разных типов.

my_tuple = (1, 2, 3, "Hello", True)

6.2.1 Операции с кортежами

  • Доступ к элементам:
    • tuple[index]: возвращает элемент по индексу.
  • Преобразование кортежа в список:
    • list(tuple): преобразует кортеж в список.
my_tuple = (1, 2, 3)

print(my_tuple[1])  # 2

my_list = list(my_tuple)  # [1, 2, 3]

6.3 Словари

Словари - это неупорядоченные коллекции данных, которые состоят из пар ключ-значение.

my_dict = {"name": "John Doe", "age": 30, "city": "Moscow"}

6.3.1 Операции со словарями

  • Добавление элементов:
    • dict[key] = value: добавляет элемент в словарь с ключом key и значением value.
  • Удаление элементов:
    • del dict[key]: удаляет элемент из словаря по ключу key.
  • Поиск элементов:
    • key in dict: возвращает True, если ключ key существует в словаре, False - otherwise.
my_dict = {"name": "John Doe", "age": 30}

my_dict["city"] = "Moscow"  # {"name": "John Doe", "age": 30, "city": "Moscow"}

del my_dict["age"]  # {"name": "John Doe", "city": "Moscow"}

print("name" in my_dict)  # True

6.4 Множества

Множества - это неупорядоченные коллекции уникальных элементов.

my_set = {1, 2, 3, "Hello", "Привет!"}

6.4.1 Операции со множествами

  • Добавление элементов:
    • set.add(element): добавляет элемент в множество.
  • Удаление элементов:
    • set.remove(element): удаляет элемент из множества.
  • Объединение множеств:
    • set1 | set2: объединяет два множества.
  • Пересечение множеств:
    • set1 & set2: находит пересечение двух множеств.
  • Разность множеств:
    • set1 - set2: находит разность между двумя множествами.
  • Симметричная разность множеств:
    • set1 ^ set2: находит симметричную разность между двумя множествами.
my_set = {1, 2, 3}

my_set.add(4)  # {1, 2, 3, 4}

my_set.remove(2)  # {1, 3, 4}

my_set1 = {1, 2, 3}
my_set2 = {3, 4, 5}

my_set3 = my_set1 | my_set2  # {1, 2, 3, 4, 5}
my_set4 = my_set1 & my_set2  # {3}
my_set5 = my_set1 - my_set2  # {1, 2}
my_set6 = my_set1 ^ my_set2  # {1, 2, 4, 5}

6.5 Строки

Строки - это неизменяемые последовательности символов.

my_string = "Hello, world!"

6.5.1 Операции со строками

  • Конкатенация:
    • str1 + str2: объединяет две строки.
  • Поиск подстроки:
    • str.find(substring): возвращает индекс первого вхождения substring в строке.
  • Разбиение строки:
    • str.split(separator): разбивает строку на подстроки по разделителю separator.
my_string = "Hello, world!"

my_string2 = my_string + "!"  # "Hello, world!!"

print(my_string.find("world"))  # 7

my_list = my_string.split(", ")  # ["Hello", "world!"]

6.5.2 Функции и методы

  • str.upper(): Возвращает строку в верхнем регистре.
  • str.lower(): Возвращает строку в нижнем регистре.
  • str.title(): Возвращает строку, в которой каждое слово начинается с заглавной буквы.
  • str.capitalize(): Возвращает строку, в которой первое слово начинается с заглавной буквы, а остальные - с маленькой.
  • str.strip(): Возвращает строку без пробелов слева и справа.
  • str.lstrip(): Возвращает строку без пробелов слева.
  • str.rstrip(): Возвращает строку без пробелов справа.
my_string = "Hello, world!"

print(my_string.upper())  # HELLO, WORLD!
print(my_string.lower())  # hello, world!
print(my_string.title())  # Hello World!
print(my_string.capitalize())  # Hello, world!
print(my_string.strip())  # Hello, world!
print(my_string.lstrip())  # Hello, world!
print(my_string.rstrip())  # Hello, world!

6.6 Срезы

В Python срезы (slices) позволяют получать подпоследовательности элементов из последовательностей, таких как строки, списки и кортежи. Синтаксис срезов очень гибкий и позволяет выбирать элементы с определенными индексами или диапазонами индексов.

последовательность[начало:конец:шаг]
  • начало: индекс, с которого начинается срез (включительно).
  • конец: индекс, на котором заканчивается срез (не включительно).
  • шаг: шаг, с которым выбираются элементы (по умолчанию 1).

6.6.1 Использование срезов

  1. Получение подпоследовательности от начала до конца (включительно):
    my_list = [1, 2, 3, 4, 5]
print(my_list[:])  # Выводит [1, 2, 3, 4, 5]
  1. Получение подпоследовательности с определенного индекса до конца:
    my_string = "Hello, world!"
print(my_string[7:])  # Выводит "world!"
  1. Получение подпоследовательности с начала до определенного индекса (не включая его):
    print(my_string[:5])  # Выводит "Hello"
  1. Получение подпоследовательности с определенным шагом:
    print(my_list[::2])  # Выводит [1, 3, 5]
  1. Получение подпоследовательности с обратным шагом:
    print(my_string[::-1])  # Выводит "!dlrow ,olleH"

6.6.2 Использование срезов с отрицательными индексами

Отрицательные индексы позволяют выбирать элементы с конца последовательности.

print(my_list[-3:])  # Выводит [3, 4, 5]
print(my_string[:-6])  # Выводит "Hello"

6.6.3 Примечание

  • Если начало не указано, срез начинается с начала последовательности.
  • Если конец не указан, срез заканчивается в конце последовательности.
  • Если шаг не указан, выбираются все элементы с шагом 1.

7. Функции

Функции - это блоки кода, которые могут быть вызваны многократно.

7.1 Определение функций

Функция в Python - это блок кода, который выполняет определенную задачу при вызове. Она позволяет структурировать код, делая его более читаемым, и уменьшает дублирование кода, что способствует повторному использованию. В Python функции определяются с использованием ключевого слова def, за которым следует имя функции и список параметров в круглых скобках, а затем блок кода, который выполняется при вызове функции.

def my_function():
    print("Hello, world!")

my_function()  # Hello, world!

7.2 Аргументы функций

Аргументы функций - это значения, которые передаются функции при ее вызове.

def my_function(name):
    print("Hello, " + name + "!")

my_function("John")  # Hello, John!

7.2.1 Типы аргументов

  • Обязательные аргументы: Эти аргументы должны быть переданы при вызове функции.
    def my_function(name):
    print("Hello, " + name + "!")

# Ошибка: недостаточно аргументов
my_function()
  • Необязательные аргументы: Эти аргументы не являются обязательными и могут быть пропущены.
    def my_function(name, age = 18):
    print("Hello, " + name + "! You are " + age + " years old.")

my_function("John")  # Hello, John! You are 18 years old.
my_function("John", 30)  # Hello, John! You are 30 years old.
  • Позиционные аргументы: Эти аргументы передаются функции по порядку.
    def my_function(a, b):
    print(a, b)

my_function(1, 2)  # 1 2
  • Именованные аргументы: Эти аргументы передаются функции по имени.
    def my_function(a, b):
    print(a, b)

my_function(b=2, a=1)  # 1 2

7.3 Возвращаемые значения

Функции могут возвращать значения с помощью ключевого слова return. Это позволяет функции возвращать результат своей работы для дальнейшего использования в программе.

def my_function():
    return "Hello, world!"

message = my_function()
print(message)  # Hello, world!

7.4 Встроенные функции

Python предоставляет множество встроенных функций для обработки данных, выполнения математических операций, работы с последовательностями и других задач.

  • print(): Выводит сообщение на экран.
  • len(): Возвращает длину объекта.
  • type(): Возвращает тип объекта.
  • int(): Преобразует объект в целое число.
  • float(): Преобразует объект в число с плавающей запятой.
  • str(): Преобразует объект в строку.
  • list(): Преобразует объект в список.
  • tuple(): Преобразует объект в кортеж.
  • dict(): Преобразует объект в словарь.
  • set(): Преобразует объект в множество.
# Выводит сообщение на экран
print("Hello, world!")

# Возвращает длину строки
length = len("Hello, world!")

# Преобразует строку в целое число
number = int("123")

# Преобразует число в строку
string = str(123)

# Создает список
my_list = list([1, 2, 3])

# Создает кортеж
my_tuple = tuple((1, 2, 3))

# Создает словарь
my_dict = dict({"name": "John Doe", "age": 30})

# Создает множество
my_set = set({1, 2, 3})

7.5 Обработка исключений

На данном этапе возникновение ошибки, или исключения, означает крах программы, т.е. ее аварийное завершение. В реальных программах такого происходить не должно. Программа должна выявлять ошибки, обрабатывать их и продолжать работу.

В качестве примера рассмотрим программу, в которой возникает ошибка деления на 0.

def spam(divideBy):
    return 42 / divideBy

print(spam(2))
print(spam(12))
print(spam(0))
print(spam(1))

Мы определили функцию spam() с одним параметром, а затем пытаемся вывести на экран возвращаемое ею значение при различных аргументах, чтобы посмотреть, что при этом произойдет.

Исключение ZeroDivisionError возникает всякий раз, когда предпринимается попытка выполнить деление на нуль. По указанному в сообщении об ошибке номеру строки можно легко определить, что виновницей является инструкция return в функции spam().

Ошибки можно обрабатывать с помощью инструкций try и except. Если в определенном фрагменте программы потенциально может возникнуть ошибка, следует поместить этот код в блок try. В случае возникновения ошибки выполнение передается в начало блока except.
Попробуем поместить проблемную строку в блок
try и обработать соответствующее исключение в блоке except.

def spam(divideBy):
    try:
        return 42 / divideBy
    except ZeroDivisionError:
        print('Error: Invalid argument.')

print(spam(2)) # 21.0
print(spam(12)) # 3.5
print(spam(0)) # Error: Invalid argument.
                    # None
print(spam(1)) # 42.0

8. Работа с библиотеками

8.1 Установка библиотек

Для установки сторонних библиотек в Python обычно используется инструмент управления пакетами pip (Python Package Installer). Для установки библиотеки необходимо выполнить команду в терминале или командной строке:

pip install имя_библиотеки

8.2 Импорт библиотек

После установки библиотеки её можно импортировать в свой Python-скрипт или интерактивную среду разработки с помощью ключевого слова import.

import имя_библиотеки

Можно также использовать сокращенные имена для библиотек с помощью ключевого слова as:

import имя_библиотеки as имя

8.3 Использование функций библиотек

После импорта библиотеки можно использовать её функции и классы в своем коде. Для вызова функций из библиотеки используется синтаксис:

название_библиотеки.название_функции(аргументы)

Например, если у нас есть библиотека math и мы хотим использовать функцию sqrt() для вычисления квадратного корня, мы можем сделать это следующим образом:

import math

x = math.sqrt(16)
print(x)  # 4.0

В случае, если мы используем сокращенное имя библиотеки, синтаксис остается тем же, но с учетом сокращенного имени:

import math as m

x = m.sqrt(16)
print(x)  # 4.0
