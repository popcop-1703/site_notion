1.4.10 History

Цель работы:

  • Разобраться как работает history.
  • Научиться работать с утилитой.

Защита:

  • Необходимо проделать все шаги очно.
  • Оформить отчёт о проделанной работе.

2.9.1. Повторение последней команды

Чтобы повторить последнюю команду в bash, введите !!.

2.9.2. Повторение других команд

Вы можете повторить другие команды, используя один !, за которым следует один или несколько символов.

Оболочка повторит последнюю команду, которая начиналась с этих символов.

2.9.3. История команд

history

history 50

2.9.4. Повторение команд через номер

При вводе текста ! за которым следует номер, предшествующий команде, которую вы хотите повторить, затем оболочка повторит команду и выполнит ее:

!561
!573

2.9.5. Сtrl+r для поиска в истории

Использование ctrl+r для поиска в истории. На скриншоте ниже я набрал только ctrl+r, за которым следуют четыре символа ls, и он находит последнюю команду, содержащую эти четыре последовательных символа:

ctrl+r

2.9.6. Переменная $HISTSIZE

Переменная $HISTSIZE определяет количество команд, которые будут запоминаться в
вашей текущей среде. В большинстве дистрибутивов по умолчанию эта переменная равна
500 или 1000:

echo $HISTSIZE

2.9.7. Переменная $HISTFILE

Переменная $HISTFILE указывает на файл, содержащий вашу историю. Оболочка bash по умолчанию устанавливает это значение равным ~/.bash_history:

echo $HISTFILE

2.9.8. Переменная $HISTFILESIZE

Количество команд, хранящихся в вашем файле истории, можно задать с помощью $HISTFILESIZE:

echo $HISTFILESIZE

Чтобы изменить значение, введите:

HISTFILESIZE=15000

Закрытие gnome-терминала с помощью мыши или ввод reboot от имени root не сохранит историю вашего терминала.

2.9.9. Повторения с помощью регулярных выражений

При использовании ! (bang) для повторения команд можно использовать регулярные выражения. На скриншоте ниже 1 заменяется на 2.
