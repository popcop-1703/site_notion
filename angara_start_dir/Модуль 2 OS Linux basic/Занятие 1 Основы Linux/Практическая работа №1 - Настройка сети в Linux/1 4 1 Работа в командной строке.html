1.4.1 Работа в командной строке

Цель работы:

  • Научиться работать в командной строке на уровне опытного пользователя
  • Использовать только серверную Ubuntu

Использовать методические указания:

  1. Основы Linux #1
  1. Основы Linux #2
  1. Шпаргалки по OS Linux

Защита практической работы:

  • Выполнить практику
    • вы можете выполнить любой из вариантов
    • каждый вариант практики призван освоить командную строку для разных уровней пользователей (один человек = один вариант задачи)
  • Составить отчёт в виде инструкции
  • Ответить на вопросы

Задачи:

  1. Задача для новичка в Linux
  1. Задача для пользователя в Linux
  1. Задача для опытного пользователя в Linux
