1.1.3 uptime

Эта команда покажет, когда и сколько система работала и, соответственно, если она работала без перерыва 3 года, то злоумышленник может воспользоваться старыми уязвимостями для получения доступа или внедрения кода.

1.2 Права доступа в системе

Давайте посмотри права директории /home

ls -la /home

Чтобы ограничить права другим пользователям делаем следующее:

sudo chmod o-rx /home/kali

Разрешение к файлу для всех пользователей является очень критичным в плане безопасности, а также нигде в системе не должно быть паролей в открытом виде.

1.3 Обмен данных между компьютерами

Команда ip neigh показывает с какими компьютерами происходит обмен данными.

ip neigh

1.4 Запущенные сервисы

Показывает какие сервисы и с какими портами запущены

sudo ss -tlnp
