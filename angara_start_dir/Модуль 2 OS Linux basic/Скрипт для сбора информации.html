Скрипт для сбора информации

Исполняемый файл скрипта

Запуск скрипта

Для начала нужно выполнить следующие изменения в файле /etc/environment. Нужно добавить строку PROMPT_COMMAND=”history -a” . Это можно сделать либо при помощи текстового редактора

sudo nano /etc/environment
PROMPT_COMMAND="history -a"

Далее добавим данную переменную в текущую оболочку при помощи source /etc/environment

Либо же можно перезайти данным пользователем в систему 

source /etc/environment

Проверить переменную можно при помощи echo $PROMPT_COMMAND. В выводе должна быть строка history -a

Если не выполнить данные действия, .bash_history будет обновляться только после завершения сессии. А PROMPT_COMMAND позволяет исполнять history -a перед каждым разом, когда bash готов показать командную строку

Также изменим значение переменных HISTSIZE и HISTFILESIZE в ~/.bashrc для того, чтобы не переполнить .bash_history у пользователя от которого будеть исполняться бОльшая часть команд

После этого требуется ввести source ~/.bashrc или выйти и зайти в терминал

После этого можно запускать скрипт. Скрипт необходимо запускать от пользователя, который содержит ваше ФИО.

Скрипт необходимо запускать после каждого запуска виртуальной машины. Остальные шаги повторять не нужно.

sudo chmod +x compiled_script.sh.x
nohup sudo ./compiled_script.sh.x &
